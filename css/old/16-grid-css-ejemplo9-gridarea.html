La propiedad del área de cuadrícula

Puede usar la propiedad grid-area para especificar dónde colocar un elemento.

La sintaxis es:

cuadrícula-fila-inicio/cuadrícula-columna-inicio/cuadrícula-fila-fin/cuadrícula-columna-final.
